그린사이클 캠페인 마이크로사이트 기능 개발.
라이브러리는 jQuery, Fabric.js 사용.
Fabric.js API 및 canvas toDataURL로 페이지 내, 스티커로 화면을 꾸민 후에 캡처하여 저장하는 페이지 기능 개발.
iOS/AOS 최적화 및 OS별 분기 처리하여, 각 이슈 처리 진행.

클라이언트 측 Jenkins와 사내 gitlab 연동 및 관리 진행(서버 개발자와 직접 소통 진행).
디바이스 사이즈는 최소 320px, 최대 768px 기준으로 작업 진행(Mobile Only).
인스타그램에서 '#그린사이클' 해시태그로 확인 가능.
